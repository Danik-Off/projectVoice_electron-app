# Shared Layer

Общие компоненты, утилиты и типы, используемые во всем приложении.

## Структура

```
shared/
├── components/        # Переиспользуемые UI компоненты
├── hooks/            # Переиспользуемые хуки
├── utils/            # Утилиты
├── types/            # Общие типы
└── constants/        # Константы
```

## Принципы

- Shared не зависит от модулей
- Shared может зависеть от core
- Все компоненты должны быть переиспользуемыми
- Нет бизнес-логики в shared
