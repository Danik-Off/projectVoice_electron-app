# Core Layer

Ядро приложения - базовые сервисы и инфраструктура, которые используются всеми модулями.

## Структура

```
core/
├── api/              # API клиент и конфигурация
├── socket/           # Socket.io клиент
├── webrtc/           # WebRTC клиент
├── store/            # MobX stores (auth, theme, notifications)
├── config/           # Конфигурация приложения
├── types/            # Базовые типы
└── utils/            # Утилиты ядра
```

## Принципы

- Core не зависит от модулей
- Все модули могут использовать core
- Core содержит только базовую функциональность
- Нет бизнес-логики в core
